# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `GET /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Quotes

- `GET /api/quote`
  - custom route to retrieve one random quote

### Scores

- `POST /api/scores`
